# 🧠 AI Code Debugger

## 📌 Description
AI Code Debugger is a Python-based intelligent system that analyzes code, detects errors, and provides suggestions for fixing them across multiple programming languages.

## 🚀 Features
- Supports Python, Java, C++, JavaScript
- Detects syntax & logical errors
- Suggests fixes using AI logic
- Feedback storage system

## 🛠 Tech Stack
- Python
- Flask
- CSV (for feedback storage)

## 📂 Project Structure
- app.py → Main application
- handlers/ → Language-specific debugging logic
- utils/ → Suggestion & feedback modules
- data/ → Stores user feedback
